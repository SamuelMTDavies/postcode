## Pattern Sources

All patterns are sourced from:
1. **Primary**: Unicode CLDR (Common Locale Data Repository) v26.0.1
2. **Secondary**: National postal service websites and official documentation
3. **Verification**: Cross-referenced with ISO standards and government sources

## Notes

- **Pattern Validation**: Patterns validate format only, not existence
- **Case Sensitivity**: Most patterns work with both uppercase and lowercase
- **Spaces**: Many patterns allow optional spaces in postal codes
- **Updates**: Patterns are updated as official formats change

## Countries Without Postal Codes

Not all countries have postal code systems. Countries not listed above typically do not use postal codes or do not have defined validation patterns in the CLDR database.

Examples include:
- Hong Kong (uses building/street names)
- Macau (no postal code system)
- Many small island nations
